About

Professionally, I'm a computer technician.
Academically, I'm a computer programmer.
Personally, I'm a Linux admin.

My name is David Seguin. I've been a user of free and open source software since 2010, diving head-first into the history of UNIX and its derivatives.

I've cut my teeth on C++ writing small programs and games, before discovering the simplicity and power of C. Soon after I discovered OpenGL and SDL, which launched my journey into graphics programming and linear algebra. I've also experimented with audio programming using SDL's built-in audio interface.

As for Linux, I primarily run Debian, but I have experience with CentOS, Ubuntu, Fedora, OpenSUSE, etc. I also run a FreeNAS server, serving as bulk network storage.

I have much more to learn about programming and open source, so I'm chronicling my adventures on this site. Hopefully this information might be useful to someone.
